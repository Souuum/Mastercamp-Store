# mastercamp-store
Mastercamp Web shop project at EFREI


```
gitclone https://github.com/Souuum/mastercamp-store && cd mastercamp-store
npm i next@13.3.4
npm install blitz
npm install
```

to run the project:

```
blitz dev
```

database can be seen with prisma studio on port 5555

```
blitz prisma studio
```
